Delete Your Data

Last updated: February 25, 2026

SportTracking by SLAMA Consulting allows you to delete specific data without deleting your entire account. This page explains what data you can delete, how to do it, and when data is automatically removed.

1. Delete a Trip (Activity)

You can permanently delete any of your recorded sport activities (trips).

1
Open Trip History Tap the History tab in the bottom navigation bar.
2
Select a trip Tap on the trip you want to delete to open the Trip Detail screen.
3
Tap the trash icon In the top-right corner, tap the trash icon (🗑).
4
Confirm deletion A confirmation dialog will appear. Tap "Delete" to permanently remove the trip.

Data deleted: Trip record, all GPS track points, speed, distance, calories, elevation, and weather data associated with the trip. Deleted from both your device and our servers.

2. Delete an Event

Event creators can permanently delete events they have created.

1
Open Events Tap the Events tab in the bottom navigation bar.
2
Select your event Tap on the event you created to open the Event Detail screen.
3
Tap the trash icon In the top-right corner, tap the trash icon (🗑). This button is only visible if you are the event creator.
4
Confirm deletion A confirmation dialog will appear. Tap "Delete" to permanently remove the event.

Data deleted: Event details (title, description, location, date), participant list, and all associated notifications. Deleted from both your device and our servers.

3. Delete a Chat Message

You can delete your own messages in event group chats.

1
Open the event chat Go to an event, then tap the chat icon in the top bar to enter the group chat.
2
Long-press your message Press and hold on the message you want to delete. A menu will appear.
3
Tap "Delete" Select the Delete option from the menu.
4
Confirm deletion A confirmation dialog will appear. Tap "Delete" to remove the message.

Data deleted: The message content is marked as deleted and no longer visible to any users in the chat. You can only delete your own messages.

Reporting: To report another user's inappropriate message, long-press their message and select "Report", then choose a reason.

4. Delete Notifications

You can delete individual notifications or clear all notifications at once.

Delete a single notification:

1
Open Notifications Tap the Notifications tab in the bottom navigation bar.
2
Tap the X button On the notification you want to remove, tap the X (close) icon on the card. The notification is deleted immediately.

Delete all notifications:

1
Tap the sweep icon In the top-right corner of the Notifications screen, tap the sweep (clear all) icon.
2
Confirm A confirmation dialog will appear. Tap "Delete" to remove all notifications.

Data deleted: Notification records are permanently removed from both your device and our servers.

5. Edit Profile Data

You can modify your personal information at any time:

1
Go to Profile → Edit Tap the Profile tab, then tap the "Edit" button.
2
Modify your data You can update your name, gender, date of birth, weight, height, fitness level, and profile photo.
3
Save Tap "Save" to apply changes. Previous values are overwritten.

6. Automatic Data Deletion Automatic

Some data is automatically cleaned up by the app:

Data Type Auto-Deleted When What Is Removed
Past events After the event end time has passed The event and all its chat messages are permanently deleted from servers (checked every hour)
Event-related notifications When the associated event is deleted or expires All notifications linked to the event are removed
Reported user messages (auto-moderation) When a user receives 2+ reports in the same event All messages from the reported user in that event are automatically removed

If you want to delete your account and ALL associated data, please visit our Account Deletion page for instructions.
